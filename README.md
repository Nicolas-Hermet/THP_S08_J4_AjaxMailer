## L'objectif :
Utiliser Ajax pour simuler un affichage et suppression de mails


## versions utilisées
Ruby : 2.5.1
Rails : 5.2.1


****



## Installation en local : 
Après un clone faite : 
```
$> bundle install
$> rails db:create
$> rails s
```
Puis lancer localhost:3000 dans votre navigateur.

Enjoy :wink:
